Animal Spirits — State Feed

This repository publishes a static JSON feed representing the current composed field state of the Animal Spirits observatory.

The feed is refreshed every 15 minutes by a scheduled GitHub Action that fetches from Twelve Data (market), FRED (macro stress), Wikimedia Pageviews (attention), and GDELT (narrative), then composes a unified state.

{
  "timestamp": "ISO-8601 UTC",
  "regions": {
    "us":    { "attention": number|null, "market": number|null, "narrative": number|null },
    "uk":    { ... },
    "india": { ... }
  },
  "meta": {
    "attention": "live" | "simulated",
    "market":    "live" | "simulated",
    "narrative": "live" | "simulated"
  }
}

All scalars are in approximately [−1, +1]. Negative values indicate stress/contraction-coded states; positive values indicate expansion-coded states.

See the repository README for the full normalisation methodology.
